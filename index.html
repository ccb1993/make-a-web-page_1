Important Concepts

Lesson 1: Web Basics & HTML

World Wide Web: How it Works
  • Many computers communicate with eachother
  • When someone goes to a web page, an HTTP Request is sent to a server (computer at another location hosting web files)
  • Servers send back appropriate HTML documents to fulfill requests sent by user computers
  • Computers interpret HTML documents via browsers
HTML
"Hyper Text Markup Language"
  • HTML documents are the basis of the web and form most web content
  • text content is "what you see"
  • markup is "what it looks like"
  • used to reference other documents and provide links to other pages
  • regular text in HTML will appear as just regular text
Tags and Elements
  • HTML documents consist of HTML elements
  • Elements consist of tags (opening at the beginning, closing at the end) which define the properties of text inside HTML documents
  • example of a tag: "em" to make text italicized
Computers are Stupid
  • They interpret instructions literally, so small mistakes can cause huge problems. They don't have deductive reasoning skills like people (not always).
Basic Structure - Inline vs. Block Elements
  • Inline elements stay in line
  • Block elements form a "box" around what is inside them making the content a package deal with regards to where it is located on a page

Lesson 2: Creating Structure with HTML

Discovering Strucutre of Webpages using Developer Tools
  • Developer tools can be accessed from the browser
  • HTML elements can be seen in blocks (boxes). Blocks form around whatever element is inside an HTML document
  • Style can be seen in the styles section on the right-hand side of developer tools
  • by clicking through the triangles on the left-hand side of Developer tools, you can see most elements in an HTML page
Hint: you can right-click and click "inspect element" to automatically access Developer Tools
"Tree Structure" of HTML
  • Elements fall under elements in html
  • the more indented an element is, the more ingrained into a block it is
Text Editors
  • can use programs like sublime to automatically create tags and easily correspond two languages in one document (e.g. add css styling to html document)
  • can link a styling language directly inside an html document by using "head" tag
Box Sizing and Positioning
  • HTML elements are boxes with 4 components
  • Sizing: each component of an "element box" has a certain measurement in pixels which changes depending on the web page
  • Positioning: div elements take the whole length of a page in line (using certain elements like display:flex it is possible to change this property)

Lesson 3: Adding Style to HTML with CSS

Avoiding Repetition
Avoiding repetition is very important!!
  • Avoiding Errors: When programmers use repetition too much, they have to do the same task over and over. This means that if they need to replace one element with another, they need to go into their HTML documents replacing each element (Tedious tasks like this leave room for error if they miss one change). If you avoid repetition, you don't need to worry as much about replacing code every single place it appears.
  • Consistency:By avoiding repetition, it makes it easy for programmers to have one change be applied to multiple places at once. For example, in this assignment, I have used class = "lesson" so that when I make design changes to anything under "lesson" it is applied across the entire document as opposed to individually changing each section if they didn't have class = "lesson".
Strategies for coding CSS
  1. look for natural boxes
  2. look for repeated styles and semantic elements
  3. write your html
  4. apply styles (biggest to smallest)
  5. fix things
What does CSS Stand for?
Cascasing Styling Sheets
  • Cascading = most specific rules are those applied at the highest level as well
  • Cascading implies inheritance meaning that properties are not only applied to a specific element, but also its descendents
  • Descendent elements (those which are more indendented) inherit any CSS property value from any ancestor element enclosing them
